# decaytracker

> The Trust Feed — AI-powered public feed of URL trust audits. Paste any URL (Amazon, App Store, any website) → AI agent scrapes, analyzes reviews, detects fake reviews/dark patterns/enshittification → publishes trust score to public feed. Companies accumulate audits and form trust ratings. Brutalist Mono terminal design.
